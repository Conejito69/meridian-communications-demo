# 5-Minute Walkthrough

**Goal:** prove enterprise-level Salesforce judgment fast. No scrolling
through Setup menus — every second shows something, not navigation.

## If they interrupt with a question

Common ones and where the real answer lives, so you don't freeze:

- **"Why no Apex?"** → It's a deliberate constraint, not a limitation —
  every automation need in this project had a clean declarative solution.
  (Don't over-explain unless asked to go deeper.)
- **"Is this real data?"** → Fully synthetic, generated by a script, ~150
  records across the model — see `../PORTFOLIO/sample-data/README.md` if
  they want the exact breakdown.
- **"How long did this take?"** → Answer honestly based on your actual
  timeline — this doc doesn't script a fake answer for you.

## What NOT to do in 5 minutes

Don't open Object Manager and scroll fields. Don't explain every Permission
Set. Don't apologize for anything ("sorry this is just a demo org") — it's
not just a demo org, it's a complete architecture, say so.
